Image Studio · Strategic Force Assessment

Generate all skins. Deploy in place.

Loads every skin's prompts, generates concept art through Google Gemini, and either downloads a ZIP organised into the correct subdirectories, or pushes it straight into your live Netlify site.

1 · Gemini API key

Get one at aistudio.google.com/apikey. Free tier covers all four skins. Stored only in this browser when you click Save.

2 · Skins to generate

Each skin's prompts are loaded automatically when the page opens. Tick the skins you want. Each runs end-to-end before the next starts.
ms between API calls — 1200 is polite

3 · Generate

Click Generate everything to run the full batch for the selected skins. Generate missing only retries failures and skips already-finished images. The grid below fills in as images return.
—

4 · Sync to Netlify

Pushes code updates and newly generated images as a single atomic deploy. Files already on Netlify that you didn't change — including previously-deployed images — are kept automatically. Only changed files are re-uploaded.
Get one at app.netlify.com/user/applications → New access token. Used only to deploy from this browser; rotate after use.
A · Drop a deploy ZIP (no terminal, recommended). The Studio unzips it in your browser, hashes every file, and pushes only what changed.
B · Skip (terminal route): if you opened this page from localhost:8000 via python3 -m http.server, the Studio reads sibling files directly. From the live URL, this fetches the already-live versions, so code wouldn't actually update.
No ZIP loaded — sync will use the live-site code as the baseline.
